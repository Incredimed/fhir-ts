# fhir-ts
FHIR TypeScript definitions generator.

## Installation
```sh
npm install fhir-ts -g
```

## Usage
```sh
fhir-ts <pattern> <output-directory>
```

Example:
```sh
fhir-ts "structure-defintions/**.profile.json" "types"
```
